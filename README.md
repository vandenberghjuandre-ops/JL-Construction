## Deploy to Netlify (2 methods)

### Method A — Drag & Drop (Easiest, no account needed)
1. Install Node.js from nodejs.org
2. Open terminal in this folder
3. Run: `npm install`
4. Run: `npm run build`
5. Go to netlify.com → sign up free
6. Click "Add new site" → "Deploy manually"
7. Drag the `dist` folder onto the page
8. Done — your app is live!

### Method B — GitHub Auto-Deploy (Best for updates)
1. Push this folder to a GitHub repository
2. Go to netlify.com → "Import from GitHub"
3. Select your repo — Netlify auto-detects the build settings
4. Click Deploy — live in 2 minutes
5. Every time you push to GitHub, Netlify auto-updates your site

## Custom Domain
1. Buy jlconstruction.co.za at domains.co.za
2. In Netlify → Site settings → Domain management → Add custom domain
3. Follow DNS instructions — takes ~10 minutes
